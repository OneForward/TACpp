### 任务描述



用结构体表示一个复数，编写实现复数的加法、乘法、和输出的函数。

* 加法规则：$$(a+bi)+(c+di)=(a+c)+(b+d)i$$
* 乘法规则：$$(a+bi)×(c+di)=(ac-bd)+(bc+ad)i$$
* 输出规则：如果$$a$$是实部，$$b$$是虚部，输出格式为$$a+bi$$


### 测试说明

输入描述：四个整型 $$a$$, $$b$$, $$c$$, $$d$$ ，分别是复数 $$x = a + bi$$ 和 $$y = c + di$$

输出描述：

将依次输出

```
x = ?
y = ?
x += y; x = ?
y *= x; y = ?
x + y = ?
y * x = ?
x = ?
y = ?
```

样例输入1：`1 2 3 4`

样例输出1：

```cpp
x = 1+2i
y = 3+4i
x += y; x = 4+6i   
y *= x; y = -12+34i
x + y = -8+40i     
y * x = -252+64i   
x = 4+6i
y = -12+34i 
```
